# Rum & Popcorn

A personal blog and portfolio site built with modern web technologies.

## Tech Stack

- **Astro** - Static site generator for fast, content-focused websites
- **Tailwind CSS** - Utility-first CSS framework for styling
- **MDX** - Markdown with JSX support for rich blog content
- **TypeScript** - Type-safe JavaScript

## Features

- **Blog Posts** - Technical writing, tutorials, and personal insights
- **Project Showcase** - Portfolio of personal projects and work
- **RSS Feed** - Subscribe to updates via RSS
- **Responsive Design** - Mobile-first, accessible interface
- **Dark Mode** - Comfortable reading in any lighting

## Structure

```
src/
├── components/     # Reusable UI components
├── content/        # Blog posts and project descriptions
├── layouts/        # Page templates
├── pages/          # Route definitions
└── styles/         # Global styles and Tailwind config
```

## Development

Built with modern tooling and best practices:
- Component-based architecture
- Type safety with TypeScript
- Fast builds and hot reload
- Optimized for performance and SEO
